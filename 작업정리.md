## 기술스택
- Next.js 16.1.6 (App Router) + React 19 + TypeScript
- Tailwind CSS 4 (`@theme inline` 방식, tailwind.config 없음)
- `@tailwindcss/postcss` 플러그인 (postcss.config.mjs)
- Pretendard Variable (CDN) - 한글 본문
- Cormorant Garamond (Google Fonts) - **영문 전용 세리프** (Luce, Real Results, Voices 등)
- 개발 서버 포트: 3002 (.claude/launch.json의 beauty-dev)

## 파일 구조
```
src/
├── app/
│   ├── globals.css          ← 전체 스타일 (컬러, 애니메이션, 마키, 글래스 등)
│   ├── layout.tsx           ← 루트 레이아웃 (폰트 CDN, SEO 메타)
│   ├── page.tsx             ← 메인 페이지 (8개 섹션)
│   ├── about/page.tsx       ← 소개 페이지
│   ├── contact/page.tsx     ← 상담 예약 페이지
│   ├── doctors/page.tsx     ← 의료진 소개
│   ├── gallery/page.tsx     ← 전후 갤러리
│   └── treatments/page.tsx  ← 시술 안내
├── components/
│   ├── Header.tsx           ← 사이드 드로어 모바일 메뉴, 세리프 Luce 로고
│   ├── Footer.tsx           ← espresso 다크 푸터, 3열 레이아웃
│   ├── MobileBottomCTA.tsx  ← 모바일 하단 고정 CTA (전화/예약)
│   ├── ScrollAnimation.tsx  ← IntersectionObserver 애니메이션 (up/left/right/scale/blur/clip)
│   ├── ReviewMarquee.tsx    ← 2줄 마키 자동스크롤 리뷰
│   ├── BeforeAfterSlider.tsx← 시술 전후 비교 슬라이더
│   ├── ContactForm.tsx      ← 상담 폼 (alert() 시연용)
│   └── ScrollReveal.tsx     ← ⚠️ 미사용 (구버전, import 안됨 - 삭제 가능)
└── data/
    ├── treatments.ts        ← 시술 데이터 (6개)
    ├── doctors.ts           ← 의료진 데이터 (3명)
    └── reviews.ts           ← 리뷰 데이터
```

## 메인 페이지 (page.tsx) 섹션 구성 — 4차 리디자인 후
1. **Hero** - 풀스크린 배경 이미지 + 중앙 텍스트 오버레이, CTA 버튼 2개
2. **Treatments** - 3열 그리드 (6개 시술 카드), 호버 시 scale 효과
3. **Doctors** - 3열 그리드 (3명 의료진 카드), object-top으로 얼굴 표시
4. **Reviews** - 마키 자동스크롤 2줄 리뷰 카드
5. **Location + Booking** - Google Maps iframe (강남역) + 예약 안내

## 애니메이션 시스템 — 4차 리디자인 후
- ScrollAnimation 컴포넌트 (IntersectionObserver, threshold 0.12)
- **fade-up 단일 애니메이션으로 통일** (기존 6종 → 1종)
- 딜레이: fade-up-d1 ~ fade-up-d5
- 마키: marquee-left / marquee-right (45s 무한 반복)
- 호버 시 마키 일시정지
- 서브페이지는 기존 type 파라미터 남아있으나 실제로는 fade-up만 동작
