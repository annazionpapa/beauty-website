## 2. 기술 스택

| 구분 | 기술 | 버전 |
|------|------|------|

### 의존성 특이사항
- 외부 라이브러리 **0개** — Next.js + React + Tailwind만 사용
- 모든 애니메이션: CSS(fade-up, marquee) + IntersectionObserver
- 아이콘: SVG 인라인 (라이브러리 미사용)
- tailwind.config 없음 — `@theme inline`으로 globals.css에서 직접 정의

### 톤앤매너
- **키워드**: 고급스러운, 절제된, 따뜻한, 전문적
- 로고: 영문 세리프 이탤릭("Luce") + 한글 산세리프("피부과")

- CTA: pill 버튼 (rounded-full), Plum 컬러
- 이미지: AI 생성(DALL-E) — 의료진, 시설, 시술 장면, B&A

### ⚠️ 폰트 규칙
- `var(--font-serif)` (Cormorant Garamond): **영문에만 사용** (한글 적용 시 Georgia 폴백으로 깨짐)
- 한글 제목: Pretendard font-light, 강조는 text-plum 색상으로
- 영문 포인트: "Luce", "Real Results", "Voices" 등 세리프 이탤릭

### 반응형
- 모바일: ~1023px (하단 고정 CTA, 사이드 드로어, body padding-bottom 64px)
- 데스크탑: 1024px~ (수평 네비, 그리드 레이아웃)
- 컨테이너: `max-w-7xl mx-auto px-6 lg:px-8`
- `prefers-reduced-motion` 대응 (애니메이션 비활성화)

## 6. 주요 기능 상세

### 상담 폼 → 시술 자동선택
- 시술 페이지에서 "상담하기" 클릭 시 `/contact?treatment=리프팅`으로 이동
- `ContactForm`에서 `useSearchParams()`로 파라미터 읽어 드롭다운 자동선택
- `treatmentParamMap` 객체로 URL 파라미터 ↔ 드롭다운 옵션명 매핑
- **필수**: `<Suspense>` 래핑 (Next.js 정적 생성 요구사항)

### 후기 시스템
- `src/data/reviews.ts`에 12건 데이터
- 홈: 12건 그리드 표시 + "후기 더보기" 링크
- 후기 페이지: 마키 무한스크롤 + 전체 리스트

### 의료진 활동사진
- 각 의사 프로필에 `activityImages` 배열 (4장씩)
- 의료진 페이지에서 2×2 그리드로 표시 (레이블 오버레이)

### Before/After
- 시술 데이터에 `beforeImage`, `afterImage`, `beforeAfterInfo` 포함
- 시술 상세 영역에서 좌우 대비 표시

## 8. 트러블슈팅

### useSearchParams() Suspense 오류
- **증상**: `npm run build` 시 "useSearchParams() should be wrapped in a suspense boundary at page /contact"
- **원인**: Next.js App Router에서 정적 생성 시 Suspense 필수
- **해결**: `contact/page.tsx`에서 `<ContactForm />`을 `<Suspense fallback={null}>`로 래핑
- **교훈**: useSearchParams 쓰는 컴포넌트 → 항상 Suspense 래핑 (모든 프로젝트 공통)

### 한글 세리프 폰트 문제
- **증상**: 한글 세리프 폰트가 피부과 톤과 안 맞음
- **해결**: 한글은 Pretendard(산세리프), 영문 포인트에만 Cormorant Garamond 사용
- **교훈**: 한글 세리프는 대부분 어색함, Pretendard가 안전한 선택

### AI 이미지 의료진 균형
- **이슈**: AI 생성 의료진 사진 얼굴 크기가 제각각
- **해결**: `imageScale` 속성 추가하여 개별 조정
- **교훈**: AI 의료진 이미지는 scale 보정 필요
